## ssh-bookmarks

A dead simple cli ssh server list.


## Installation
```zsh
$ gem install ssh-bookmarks
```
## Updating

    $ gem update ssh-bookmarks

## Uninstalling

    $ gem uninstall ssh-bookmarks

## Usage

    $ ssh-bookmarks


## Configure

    $ man ssh_config


You can use additional options in your ~/.ssh/config file
